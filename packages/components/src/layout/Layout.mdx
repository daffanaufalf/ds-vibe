# Layout Foundation

Foundational components for building consistent, token-driven layouts.

## Box

The `Box` is the most basic building block. It provides a way to apply spacing and styling using design tokens via a polymorphic `as` prop.

## Stack

The `Stack` component is used to manage the layout of its immediate children in a linear way (either horizontally or vertically). It handles the spacing between items automatically using the `gap` prop.

## System Props

Layout components support several common props for spacing and styling:

- **Spacing**: `p`, `m`, `px`, `py`, `mx`, `my`, etc. (Use CSS variables like `var(--ds-spacing-4)`).
- **Rounding**: `borderRadius` (Use `var(--ds-radius-md)`).
- **Elevation**: `shadow` (Use `var(--ds-shadow-sm)`).
- **Polymorphism**: Use `as="section"`, `as="nav"`, etc.
