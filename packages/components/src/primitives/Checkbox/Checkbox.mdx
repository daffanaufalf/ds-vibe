# Checkbox

A control that allows the user to select one or more options from a set. Built using Radix UI for maximum accessibility.

## Features

- **Radix UI Primitive**: Inherits all standard Radix Checkbox behaviors.
- **Labeling**: Built-in support for labels and accessibility pairing.
- **Validation**: Support for error messages and mandatory states.
- **Indeterminate State**: Visual support for hierarchical selection.

## Accessibility

- **Keyboard**: Use `Space` to toggle the checkbox.
- **Screen Readers**: Correctly identifies as a checkbox and announces label/description.
- **Indicator**: Visual checkmark is accessible and inherits color correctly.
