# Switch

A control that allows the user to toggle between checked and uncontrolled states.

## Features

- **Radix UI Primitive**: Full support for accessibility and interactions.
- **Visual Feedback**: Smooth transitions and clear focus indicators.
- **Labels**: Built-in support for semantic label pairing.

## Accessibility

- **Keyboard**: Use `Space` or `Enter` to toggle the switch.
- **Screen Readers**: Identifies as a switch and announces checked state.
- **Contrast**: The toggle background and thumb have high contrast for clarity.
