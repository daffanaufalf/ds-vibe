# Radio Group

A set of checkable buttons—known as radio buttons—where no more than one button can be checked at a time.

## Features

- **Radix UI Primitive**: Robust implementation of WAI-ARIA Radio Group pattern.
- **Accessibility**: Automatic arrow key navigation and label pairing.
- **Layouts**: Support for vertical and horizontal arrangements.
- **Validation**: Integrated group-level error messaging.

## Accessibility

- **Keyboard**: Use arrow keys (`Up`, `Down`, `Left`, `Right`) to navigate between items.
- **Screen Readers**: Announces the group label and the checked state of each item.
- **Polymorphism**: The labels are automatically associated with their respective inputs via generated IDs.
